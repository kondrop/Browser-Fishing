## 技術スタック（予定）

| カテゴリ | 技術 |
|---------|------|
| フロントエンド | Phaser 3 + TypeScript + Vite |
| バックエンド | Node.js + Express |
| リアルタイム通信 | Socket.io または Colyseus |
| データベース | PostgreSQL または MongoDB |
| 認証 | Firebase Auth または 自前JWT |
| ホスティング | Vercel (フロント) + Railway/Render (バック) |

## 進め方

1. **フェーズ 6-1**（クエスト）を仕上げる → フェーズ 5（ワールド）へ
2. 各フェーズ完了後にテスト＆調整
3. フェーズ 7 (ビジュアル) は並行して進めてもOK
4. フェーズ 8 (マルチプレイ) は他が安定してから着手
