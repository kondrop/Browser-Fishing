# UI画像

このディレクトリには、UI要素で使用する画像を配置します。

## ファイル一覧

### `book-texture.png`
- **用途**: 図鑑UI（`book-container`）の背景テクスチャ
- **サイズ**: 任意（シームレスにリピートできる画像）
- **形式**: PNG（透明部分があっても可）
- **使用方法**: CSSで`background-repeat: repeat`として使用されます

## 画像の要件

- **シームレス**: 上下左右がつながるようにデザインしてください
- **タイル可能**: リピートしても違和感がないパターンにしてください
- **色調**: 背景色`#e8cfa6`と調和する色合いにしてください
