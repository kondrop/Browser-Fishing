# 生息地背景画像

このディレクトリには、図鑑の詳細画面で使用する生息地別の背景画像を配置します。

## ファイル名

以下の3つの画像ファイルを配置してください：

- `freshwater-bg.png` - 淡水の背景画像
- `saltwater-bg.png` - 海水の背景画像  
- `stream-bg.png` - 渓流の背景画像

## 画像サイズ

推奨サイズ: 305px × 165px（`book-detail-image-container-new`のサイズに合わせてください）

## 使用方法

画像は自動的に生息地に応じて表示されます：
- 淡水（FRESHWATER）の魚 → `freshwater-bg.png`
- 海水（SALTWATER）の魚 → `saltwater-bg.png`
- 渓流（STREAM）の魚 → `stream-bg.png`
