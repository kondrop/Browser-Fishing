## 指示例

```
「5-10 を 1.0 にして」 → ゲージ増加速度を0.8から1.0に変更
「5-15 を 0.02 にして」 → 魚のなめらかさを0.03から0.02に変更
「5-7 を 1.2 にして」 → 重力を1.8から1.2に変更（ふわふわに）
「4-3 を 2 にして」 → 合わせの猶予時間を1.5秒から2秒に変更
```
